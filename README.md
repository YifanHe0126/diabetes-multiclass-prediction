# Diabetes Multi-class Prediction

## Overview
A machine learning project for multi-class diabetes prediction using large-scale health survey data.

## Objective
The project focuses on classifying non-diabetes, prediabetes, and diabetes cases while handling class imbalance and improving result interpretation.

## My Contribution
- Performed data preprocessing and imbalance handling
- Compared multiple classification models
- Evaluated results with precision, recall, F1, confusion matrix, and t-SNE
- Analyzed why prediabetes remained difficult to classify

## Tech Stack
Python, Scikit-learn, Machine Learning, Data Analysis
